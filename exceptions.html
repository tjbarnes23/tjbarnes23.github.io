7. Exceptions

1.
Class Name Exceptions

The Class Name for the following Methods is omitted even though these are Bob Methods under Section 4: Grandsire, Double Grandsire, Reverse Grandsire, Little Grandsire, Union, Double Union and Reverse Union. These Method Names may not be used again for either new Bob Methods or new Methods with no Class Name.

2.
Method Name Exceptions

The Method known as Little Bob has no Method Name since Little Bob is its Method Class. This Method will continue to be known as Little Bob. The most likely Method Name for this Method, were it to be given one, is ‘Plain’, and so the name ‘Plain’ may not be used for any new Little Bob Method.

3.
Doubles Variations

Doubles Variations are separate names given to Doubles Methods to indicate a specific Call or Calls with which a Doubles Method is rung.

Doubles is the only Stage at which Variations are used. At all other Stages, using different Calls does not alter the name of the Method being rung.

The CRO maintains a library of Doubles Calls that have been used to name Doubles Variations. A name given to a Doubles Call is unique across all other Doubles Calls.

The CRO also maintains a library of Doubles Variations, which cross-references to the underlying Methods Library and Doubles Calls Library.

A Doubles Variation is capable by itself of producing a single Extent.

Doubles Variations have the same Method Class as the underlying Doubles Method on which they are based.

New Doubles Variations may be named by ringing them to at least an Extent.

A name given to a Doubles Variation is unique across both Doubles Methods and Doubles Variations with a given Class Name, including being unique for Doubles Methods and Doubles Variations that do not have a Class Name.

A name given to a Doubles Variation also meets the requirements for Method Extension and Contraction shown in Section 5.C, whereby the Variation’s underlying Method is assessed versus other Stages.

Doubles Variations are incorporated into Compositions in the same way as Methods.

Doubles Variations are included in Performance Reports in the same way as Methods.
