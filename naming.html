5. Method Naming

A.
Principles
1.
Method Library

The full title is the Central Council Method Library. This is the definitive record of methods that have been rung and named, which is maintained and made available by the Central Council.

2.
Method Title

Methods recorded in the Method Library have a unique tripartite Method Title consisting of: Name, Class Descriptor, Stage Name.

Further explanation: The Class Descriptor may contain more than one term, or in some cases, notably Principles and historic methods like Grandsire, no term, i.e. the Class Descriptor is null, as defined in the next section.

3.

Methods recorded in the Method Library may not have names that conflict with the criteria related to method extension (see Section 8) or symmetry relationships (see Section 5.C).

4.

Methods recorded in the Method Library that have in the past been known by a different name and/or title will have this recorded, where details are known.

B.
Class descriptors
1.
Class descriptor

The Class Descriptor is formed by concatenating up to four terms as follows, and in the order shown:
a) Add ’Jump’ if the Method has this Class;
b) Add ‘Differential’ if the Method has this Class;
c) Add ’Little’ if the Method has this Class;
d) Add one of the following if it applies: ‘Place’, ‘Bob’, ‘Treble Bob’, ‘Surprise’, ‘Delight’, ‘Treble Place’, ‘Alliance’;
e) Otherwise leave blank.

C.
Naming constraints
1.

The following constraints apply to a new Method to be added to the Method Library.

2.

A new Method may not use the same name as a Method recorded in the Method Library if that would result in its Method Title being the same.

3.

The name given to a new Method that is recorded in the Method Library may not include Class Descriptor term(s) such that the resulting Method Title is ambiguous as to the Class of the Method.

Example: Naming a Bob Method “Quavo Treble” would give a misleading Method Title, e.g. “Quavo Treble Bob Major”. It's unclear from the Method Title whether the Method is a Bob Method or a Treble Bob Method.

4.

Rotations of a method recorded in the Method Library are not normally recorded as separate methods.

5.

The prefixes ‘Reverse’, ‘Single’ and ‘Double may only be used to begin a Method Name if the relevant symmetry relationship is satisfied (see below), in which case their use is optional.

6.

A Method Name may only begin with ‘Reverse’ if:
a) Another Method in the Method Library has the same Method Title (after excluding ‘Reverse’ in the new Method); and
b) That other Method's Changes are the Reverse of the new Method's -- that is, the Places within each Change are inverted.

Example: St Simon's Bob Doubles has the place notation 5.1.5.3.5.3.5.1.5.125.
If each place notation is inverted, the result is 1.5.1.3.1.3.1.5.1.145. This method (usually rotated to be rung as 3.1.5.1.145.1.5.1.3.1) is Reverse St Simon's Bob Doubles.

7.

A Method Name may only begin with ‘Double’ if:
a) It is a Double Method -- that is, when the Places of each Change are inverted, the same Method results; and
b) If there is another Method in the Method Library that has the same Method Title (after excluding ‘Double’, or with ‘Double’ replacing ‘Single’), then this other Method has the same place notation above the Hunt Bell (or two or more coursing Hunt Bells).

Example: Double Court Bob Minor has the place notation x14x36x16x36x14x16.
If these changes are inverted, the result is x36x14x16x14x36x16. This can be rotated to give the same Changes as above.
It is therefore a Double method.
There is also a Method called Single Court Bob Minor in the Method Library. It's place notation is x14x16x16x16x14x16.
Because the places above the hunt bell in Single Court Bob Minor are the same as in Double Court Bob Minor, Double may be used.

8.

A Method Name may only begin with ‘Single’ if:
a) It is not a Double Method; and
b) If there is another Method in the Method Library that has the same Method Title (after excluding ‘Single’, or with ‘Single’ replacing ‘Double’), then this other Method has the same place notation above the Hunt Bell (or two or more coursing Hunt Bells).

Example: See the example of Single Court Bob Minor and Double Court Bob Minor in 7. above.

9.

If one Extension Process has been used to relate two or more Methods that share a Name in the Method Library then:
a) A new Method at a different Stage that shares the same relationship shall use the same Name; and
b) A new Method at a different Stage that does not share the same relationship shall not use the same Name, even if it shares a relationship with one of the Methods based on a different Extension Process.

Example: Yorkshire Surprise Royal is related to Yorkshire Surprise Major by an Extension Process, hence these Methods may share a Name.
Southwark Surprise Maximus is also related to Yorkshire Surprise Major by a different Extension Process. Since the extension process for the name Yorkshire has already been established by the naming of Yorkshire Surprise Royal, Southwark could not have been named Yorkshire Surprise Maximus.

10.

If a new method is related to more than one method in the Method Library by different Extension Processes, and none of the existing Methods are related by an Extension Process to any other Method in the Library, then any of the valid Extension Processes may be used to name the new Method, but consideration should be given to:
a) The number of stages at which the Extension Process would create valid Methods; and
b) The number of Desirable Invariants that are preserved (see Section 8.B).

4.

Each bell is rung continuously by the same person or persons.

5.

Physical aids to memory in ringing and conducting are not used.

6.

No assistance in the execution of the ringing, such as making calls and detecting or correcting errors, is given to any ringer by any person not ringing in the Performance.

7.

Tower bells (or simulations thereof) are rung full circle-style, and handbells (or simulations thereof) are rung in alternating up-strokes and down-strokes.

8.

On handbells, every ringer rings at least two bells.

9.

Arrangements are made for interested people to be able to listen to the Performance.

10.

The ringing is heard, and the figures of the Composition are checked throughout, by competent umpire(s).

11.

The organizer of a Record Length provides notice of the attempt to The Ringing World such that the notice may be published at least 14 days prior to the attempt. The notice states the location, date and start time of the attempt, the Length and the Method(s) to be rung. The organizer also provides a copy of this notice to the CRO’s Technical and Taxonomy Workgroup at least 14 days prior to the attempt.

12.

A Performance Report (see Section 6) for a Record Length is sent to both The Ringing World for publication, and to the CRO’s Technical and Taxonomy Workgroup.

B.
Method Naming
1.

To be eligible to be recorded in the Central Council's Method Library, a new Method is rung in a Performance that either comprises at least 1250 Changes, or contains the Extent of the Method.

2.

The name given to a new Method that is recorded in the Central Council's Method Library is unique for its Class Name and Stage, including being unique for Methods that don’t have a Class Name.

3.

The name given to a new Method that is recorded in the Central Council's Method Library may not include Class Name term(s) such that the resulting Method Title is ambiguous as to the Class of the Method.

4.

The name given to a new Method that is recorded in the Central Council's Method Library follows the requirements for Method Extension and Contraction shown in Section 5.C.

5.

The Performance Report (see Section 6) for the Performance that qualifies a new Method to be recorded in the Central Council's Method Library is published in the Ringing World.

B.1.
Construction of Class Name (This section to be re-worded / re-numbered)
1.
Class Name

Class Name is constructed by concatenating up to four terms as follows and in the order shown:
1. Add the term Jump if the Method is part of this Class;
2. Add the term Differential if the Method is part of this Class;
3. Add the term Little if the Method is part of this Class; and
4. Add one of the following classification terms if it applies: Place, Bob, Slow Course, Treble Bob, Surprise, Delight, Treble Place, Alliance.
If none of the above terms applies, the Method has no Class Name.

Further explanation: In the classification diagram in A.1 above, nodes on the tree that generate a word in the Class Name are shown in bold.

6.
Reverse

A new Method is not given a name that begins with "Reverse" unless:
1. Another Method exists in the Library that has the same Method Title (after excluding Reverse); and
2. That other Method's Changes are the Reverse of the new Method's -- that is, the Places within each Change are inverted.
Even when the conditions above for the use of Reverse are met, its use remains optional.

Example: A band wishes to name a new Method "Reverse Picadilly". The new Method has a Class Name of "Bob" and a Stage Name of "Minor". The new Method's Title would therefore be "Reverse Picadilly Bob Minor". The Method Name "Reverse Picadilly" is only used if there is an existing Method with the Title "Picadilly Bob Minor", and "Reverse Picadilly Bob Minor" is the Reverse of this Method.
However, even if the conditions are met for giving the new Method the name "Reverse Picadilly", the band may still choose to give the new Method a different name, without including "Reverse".

7.
Double

A new Method is not given a name that begins with "Double" unless:
1. It is a Double Method -- that is, when the Places of each Change are inverted, the same Method results; and
2. If another Method exists in the Method Library that has the same Method Title (after excluding Double, and after excluding Single from the other Method if it used), then the new Method has the same place notation above the Hunt Bell (or two or more coursing Hunt Bells) as the existing Method.
Even when the conditions above for the use of Double are met, its use remains optional.

8.
Single

A new Method is not given a name that begins with "Single" unless:
1. It is not a Double Method; and
2. If another Method exists in the Method Library that has the same Method Title (after excluding Single, and after excluding Double from the other Method if it used), then the new Method has the same place notation above the Hunt Bell (or two or more coursing Hunt Bells) as the existing Method.
Even when the conditions above for the use of Single are met, its use remains optional.

C.
Method Extension
1.
Parent Method

A Method at a given Stage from which it may be possible to derive related Methods at higher or lower Stages using extension of contraction techniques.

2.
Extension

A Method at a higher Stage than a Parent Method that is related to the Parent Method by an extension technique.

3.
Contraction

A Method at a lower Stage than a Parent Method that is related to the Parent Method by a contraction technique.

4.

An Extension or Contraction of a Parent Method is usually obtained using one of the extension or contraction techniques maintained by the CRO’s Technical and Taxonomy Workgroup, and may be given the same Method name (if this name has not already been used) as the Parent Method from which it was derived.

However, an Extension or Contraction may also be obtained using other techniques. In this case, if the Extension or Contraction is given the same Method Name as the Parent Method from which it was derived, the extension or contraction technique that was used should be described in the Performance Report for the Performance that names the Extension or Contraction (see Section N.9).

5.

Once a given extension or contraction technique has been used to obtain an Extension or Contraction that is given the same Method Name as the Parent Method from which it was derived, bands ringing other Extensions or Contractions of that same Parent Method at other Stages that were obtained using the same extension or contraction technique are encouraged to give those other Extensions or Contractions the same Method Name.

6.

A Method should not be given the same Method Name as another Method of the same Class at a different Stage if there is no known extension or contraction technique that relates them.
