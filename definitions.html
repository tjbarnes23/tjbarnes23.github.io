3. Definitions

A
Rows
A.1
Row

A sequence of numbered bells in which no bell appears more than once.

Example: 531246 is a 6-bell Row.

Further explanation: Bells are uniquely numbered in descending order of pitch. Both cardinal numbers (e.g. 1, 2, 3 ...) and ordinal numbers (e.g. 1st, 2nd, 3rd ...) are used.

When using cardinals, number 1 is used for the bell with the highest pitch, number 2 for the next highest pitch and so on. When using ordinals, bell number 2 becomes 'the 2nd', bell number 3 becomes 'the 3rd', and so on. However bell number 1 is usually referred to as 'the treble' rather than 'the 1st', and the bell with the lowest pitch is usually referred to as 'the tenor' rather than the corresponding ordinal.

Bells are numbered relative to how many are in use. For example, if a tower has eight bells but only the six with the lowest pitches are in use, these bells are numbered 1 to 6 instead of 3 to 8.

Rows are normally shown in columns so that the path of individual bells can be visualised by looking down the columns. In order to keep the columns aligned, a fixed font can be used with single characters representing each bell. Conventionally '0' (zero) is used to represent the 10th, and upper (or sometimes lower) case letters (excluding 'I', 'O' and 'X') are used for bells 11 to 33 as follows:

'E' (Eleven); 'T' (Twelve); 'A' (Thirteen); 'B' (Fourteen); 'C' (Fifteen); 'D' (Sixteen); 'F' (Seventeen); 'G' (Eighteen); 'H' (Nineteen); 'J' (Twenty); 'K' (Twenty-one); 'L' (Twenty-two); 'M' (Twenty-three); 'N' (Twenty-four); 'P' (Twenty-five); 'Q' (Twenty-six); 'R' (Twenty-seven); 'S' (Twenty-eight); 'U' (Twenty-nine); 'V' (Thirty); 'W' (Thirty-one); 'Y' (Thirty-two); 'Z' (Thirty-three)

Technical comments: While this definition of Row refers to a sequence of bells, a Row can comprise, at minimum, one bell.

This definition of Row precludes cylindrical ringing, where a bell can appear twice in some Rows and not at all in other Rows. Cylindrical ringing is considered beyond the scope of this framework -- it would likely require its own framework, with modified definitions in areas such as Truth.

A.2
Rounds

A Row in which the bells are ordered from the lowest number to the highest number.

Example: 12345678 is Rounds for an 8-bell Row.

A.3
Place

The position of a bell within a Row.

Example: In the Row 142536, the 2nd bell is in 3rd's Place.

Further explanation: Places are numbered starting with 1 for the first bell in the Row, and increasing sequentially for each next bell in the Row. E.g. the bell that is second in a Row is in Place 2.

Each bell in a Row appears in a different Place to all other bells in that Row, and therefore there are the same number of Places in a Row as there are bells in that Row.

Places are often referred to using ordinal numbers in the possessive form, so Place 2 is often '2nd's Place'. 2nd's Place is a reference to the Place the 2nd bell occupies in Rounds.

The bell in 1st's Place is often said to be leading, and the bell in the last Place of the Row is often said to be lying. E.g. in the Row 214365, the 2nd is leading and the 5th is lying.

A.4
Extent

The complete set of distinct Rows possible for a given set of bells.

Futher explanation: The number of Rows in an Extent for a given set of bells is the factorial of the number of bells in the set. E.g. for 4 bells, the Extent has 4! Rows = 1 * 2 * 3 * 4 = 24 Rows.

Technical comments: [The following may move to a later section of the framework to be more in context.] A sequence of Rows may include the Rows of an Extent one or more times. However, this definition of Extent neither implies that the Rows appear in any particular order in the sequence, nor that the Rows making up an Extent are necessarily contiguous within the sequence of Rows.

B
Stages
B.1
Stage

A property of several Method Ringing terms that indicates the number of bells participating.

Further explanation: Stage applies to the following Method Ringing terms: Rows, Changes, Methods, Compositions, [Changesequences and Rowsequences].

The Stage of a Row is the number of bells in that Row.

The application of Stage to Changes, Methods, Compositions, [Changesequences and Rowsequences] will be covered in the 'Further Explanation' sections of the definitions of those terms.

B.2
Stage Name

Names that are given to the different Stages.

Further explanation:

1 = One; 3 = Singles; 5 = Doubles; 7 = Triples; 9 = Caters; 11 = Cinques; 13 = Sextuples; 15 = Septuples.

2 = Two; 4 = Minimus; 6 = Minor; 8 = Major; 10 = Royal; 12 = Maximus; 14 = Fourteen; 16 = Sixteen.

Odd and even Stages above 16 are named with the written form of the corresponding Stage number. For example: Seventeen, Twenty-two.

C
Changes
C.1
Change

A transposition of bells from an existing Row to a new Row, where both Rows have the same Stage.

Example: A Change might transpose the bells from the Row 214365 to the Row 241356.

Further explanation: The Stage of a Change is the number of bells on which that Change operates.

Note that the same Change applied to different Rows will have the same relative effect. For example, a Change applied to the Row 1342 might produce the Row 1432. The same Change applied to the Row 4123 would produce the Row 4213.

C.1.1
Identity Change

A Change that transposes each bell in a Row to the same Place in a new Row.

Example: The Change that transposes the bells from Row 54321 to Row 54321 is an Identity Change.

C.1.2
Adjacent Change

A Change that transposes each bell in a Row to either the same Place or an adjacent Place in a new Row, but which is not an Identity Change.

Example: The Change that transposes the bells from Row 21436587 to Row 12346857 is an Adjacent Change.

Technical comments: An Adjacent Change has a minimum Stage of 2.

C.1.3
Jump Change

A Change that transposes at least one bell in a Row to a different and non-adjacent Place in a new Row.

Example: The Change that transposes the bells from Row 12345678 to Row 23415678 is a Jump Change because the treble moves from 1st's Place to 4th's Place, which are not adjacent Places.

Further explanation: When ringing heavy bells, Jump Changes are more commonly used to jump from an earlier Place to a later Place, rather than vice versa, due to the physical difficulty of reducing the time between bell strikes.

Technical comments: A Jump Change has a minimum Stage of 3.

D
Methods
D.1
Method

A defined sequence of Changes, all with the same Stage.

Example: The sequence of Changes x16x14x16x12 is the Method that has been given the name Little Bob Minor.

Futher explanation: Individual Changes and sequences of Changes may be represented using place notation, and place notation is used in the example above as well as elsewhere in this framework. Place notation is described in Section 8.

A Method is usually given a name that is used in ringing to refer to the Method's sequence of Changes. A Method (including its name) may be recorded in the Central Council's Method Library when certain requirements have been met. Section 5.B (Method Naming) describes these requirements.

A Method has the Stage of its constituent Changes.

D.1.1
Static Method

A Method whose sequence of Changes is fixed and finite.

Example: x16x14x16x12 (Little Bob Minor) is an example of a Static Method. This Method can be represented by a fixed length place notation.

Further explanation: A Static Method is used to produce a sequence of Rows by applying the first Change in the sequence to an initial Row to obtain a second Row. The second Change in the sequence is then applied to the second Row to obtain a third Row, the third Change is applied to the third Row to produce a fourth Row, and so on.

When the last Row is produced by the last Change in the sequence, the sequence of Changes is often applied again, using the last Row produced by the first application of the sequence of Changes as the starting Row for the second application of the sequence of Changes. After applying the sequence of Changes a certain number of times, the final row produced will be the same as the initial Row used.

A Static Method is usually defined such that its sequence of Changes is not divisible into two or more equal sequences.

D.1.2
Dynamic Method

A Method whose sequence of Changes is not fixed or is not finite.

Example: Dixon's Bob Minor.

Further explanation: Dixon's Bob Minor specifies that Rows are produced by successively applying the pair of Changes x16 except (a) if the Treble is leading after an x Change, replace the next 16 Change with a 12 Change, and (b) if the 2nd or 4th is leading after an x Change, replace the next 16 Change with a 14 Change.

In this Method, the sequence of Changes varies depending on the Rows produced, so using a different initial Row could produce a different sequence of Changes. This Method's sequence of Changes is therefore not fixed, and so Dixon's Bob Minor is an example of a Dynamic Method.

D.2
Rotation

A Static Method whose Changes are the same as another Static Method, except that the Changes, when considered as a cycle, start at a different point in the cycle.

Example: The Method Plain Bob Minimus has the Changes -14-14-14-12. The Method with the Changes -14-12-14-14 is a Rotation of Plain Bob Minimus.

Further explanation: More fully, this term would be 'Method Rotation' to distinguish it from other uses of rotation in Method Ringing, such as a rotated Composition.

Technical comments: When using a Method's sequence of Changes to produce Rows, the Changes may be used starting and/or finishing at any point in the sequence. For this reason, a Rotation of an existing Method is not usually separately named, because the same Changes can be produced by starting and finishing the existing Method at a different point in its sequence of Changes.

E
Cover Bells
E.1
Cover Bell

A bell that occupies a Place in a Row that is not one of the Places operated on by the Method being applied to that Row.

Example: A Doubles Method is used with 6-bell Rows, where the Method's Changes operate on Places 1 to 5, and 6th's Place contains a Cover Bell (usually the tenor).

Further explanation: Note that in a Row containing one or more Cover Bells, the numbering of the Places may be modified to exclude the Cover Bell(s). For example, in Doubles with a Cover Bell in 6th's Place, the bell in 5th's Place may be said to be lying.

Technical comments: More formally, a Cover Bell:
1. Occupies a Place in a Row that is not one of the Places operated on by the Method(s) being applied to that Row; and
2. Remains in the same Place from one Row to the next unless it is affected by a Call, or unless there is a change to a Method whose Changes operate on the Cover Bellâ€™s Place.

F
Calls
F.1
Call

An instruction to replace Change(s) from a Method with different Change(s), change the current Method to a different one, or exchange Cover Bell(s).

Example: The last Change in Little Bob Minor's sequence of Changes is '12'. This is replaced with the Change '14' when a 'bob' is called. A 'bob' is a commonly-used Call in Method Ringing.

Further explanation: Most commonly a call replaces an individual Change with another, but a call may also add Changes, remove Changes, or replace a set of Changes with a different set of Changes.

When a Call adds or removes Changes to or from a Method, these Changes are respectively added to or subtracted from the count of Changes for that Method that is used for Performance Reporting purposes.

Technical comments: More formally, a Call is an instruction to replace zero or more Changes from one or more Methods with a different set of zero or more Changes, or to change the current Method(s) to other Method(s). Calls may also operate on Cover Bell(s) to produce variable cover ringing.

When a Call affects the Changes of more than one Method (i.e. in spliced), the Composition should make clear how the Change count of each Method is affected in order to enable accurate Performance Reporting.

G
Compositions
G.1
Composition

An arrangement of Method(s) and Call(s), all operating on Rows with the same Stage.

Further explanation: In addition to using a Method's whole sequence of Changes in a Composition, any subset of a Method's sequence of Changes may also be used.

A Composition has the Stage of the Rows it produces.

Technical comments: A Composition may also specify Cover Bells if the Composition uses variable Cover Bells and/or uses Methods of more than one Stage. A Composition normally will not specify fixed Cover Bells (for example, a Composition of Grandsire Caters doesn't specify that the tenor is always in 10th's place). However, in certain cases, such as a Composition of Doubles on the front 5, a fixed Cover Bell in 6th's place, and Minor on the back 6, the Composition could include fixed Cover Bells.
