8. Place Notation

[The description below to be further expanded.]

Changes are often represented using place notation. This notation is formed by writing the Places of bells that remain in the same Place from one Row to the next, working from beginning to end in the Row. All remaining bells swap Places with the bell next to them.

E.g. for 4-bell Changes, the place notation ‘14’ indicates that the leading and lying bells stay in the same Place across Rows, and the bells in 2nd's and 3rd's Places swap. So if Change '14' is applied to Row 2143, the resulting Row is 2413.

In place notation, single character representations of bells are used for Places above 9th's Place (e.g. "9T"). The symbols 'x' or '-' are used to denote an Adjacent Change at an even Stage where no bell stays in the same Place, so all pairs of bells swap.

Dots are used to delineate between consecutive Changes where at least one bell remains in the same Place in both Changes (e.g. '14.58'). '14.58' (when referring, e.g., to 8-bell Changes) can optionally be represented as '4.5' since the '1' and '8' can be inferred.

Also note that place notation, as described above, can only be used to describe an Adjacent Change or an Identity Change, or a sequence of such Changes. However, various extensions to place notation have been proposed that also enable the description of Jump Changes.
