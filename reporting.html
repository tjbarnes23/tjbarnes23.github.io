6. Performance Reporting

1.

A Performance Report generally includes the following information, as applicable:
(1) The society for which the Performance was rung.
(2) The location where the Performance was rung.
(3) The date on which the Performance was rung.
(4) The time the Performance took to ring.
(5) The weight of the tenor for full circle-style ringing, or the number of the tenor for handbell ringing.
(6) The Length of the Performance.
(7) The Method Title(s) of the Method(s) rung.
(8) The number of Changes rung of each Method.
(9) The number of changes of Method.
(10) The name(s) of the composer(s) of the Composition used in the Performance.
(11) The names of the ringers in the Performance and the number(s) of the bell(s) each rang.
(12) The names of any umpire(s) present.
(13) Dedications and other footnotes.

2.

A Performance involving more than one Method is described as Spliced.

3.

The Report discloses if a Performance used simulated sound.

4.

The Report of a Record Length includes a report by the umpire(s).

5.

For a non-Record Length Performance, the Report discloses if an umpire was not present for a Performance by a single person.

6.

For a non-Record Length Performance, the Report discloses which, if any, of the requirements of Sections 5.A.1 – 5.A.7 were not followed, but with the following exception:
A disclosure for non-Truth is not required if:
(1) The Performance comprised
      (a) Round Block(s) with an Effective Stage of n, and
      (b) Round Block(s) with an Effective Stage of n-1;
(2) The Rows in (a) and (b) are True when considered independently using the definition of True in Section 3.J.4; and
(3) The Rows in (a) contain at least one Extent.

7.

The Report of a Record Length includes the Composition that was rung. For other Performance Reports, submitters are encouraged to include the Composition rung, or a reference thereto, when Performance Reports are submitted electronically. Ideally the reader of a Performance Report should have enough information to be able to reproduce the Rows that were rung.

8.

A Performance Report includes the specification and proposed name(s) of any Method(s) rung in the Performance that are not in the CC's Method Library.

9.

When a new Method under 8. above is an Extension or a Contraction that was derived from its Parent Method using a technique that is not included in the CRO’s Technical and Taxonomy Workgroup’s collection (see Section 5.C), and the proposed name for the Extension or Contraction is the same as the Parent Method, the Performance Report for the Performance that names the Extension or Contraction discloses the technique that was used to extend or contract the Parent Method.
